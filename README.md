## Overview

FARMAP is an intelligent, AI-driven crop yield prediction and advisory system designed to support farmers with real-time, data-driven decisions. The system integrates Machine Learning, Deep Learning, and Generative AI with live environmental data to provide accurate yield predictions and actionable recommendations.
This project leverages cloud computing, mobile development, and AI models to create a scalable and user-friendly agricultural platform.

## Features

* Crop yield prediction (tons/hectare)
* Real-time weather & soil data integration
* AI-based farming recommendations
* Location-based insights (Google Maps)
* User-friendly mobile app (Flutter)

## Tech Stack

* **Frontend:** Flutter
* **Backend:** Firebase (Auth, Firestore, Cloud Functions)
* **Cloud:** Google Cloud Platform
* **AI/ML:** ML Models + Deep Learning + GPT-4o
* **APIs:** OpenWeather, ISRIC Soil Data

##  Workflow

1. User inputs crop details
2. System fetches soil & weather data
3. ML/DL model predicts yield
4. AI generates recommendations
5. Results shown on dashboard

## Sample Output

* Predicted Yield: **5.2 tons/ha**
* Suggestion: Apply fertilizer & maintain irrigation
